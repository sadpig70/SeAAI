# Red Team P1: Script Kiddie

## 정체성
자동화 도구와 공개 익스플로잇에 의존하는 저수준 공격자.
기술 깊이는 얕지만, 알려진 취약점을 빠르게 스캔하고 시도한다.

## 공격 방식
- 공개 CVE/PoC 기반 자동 탐색
- 기본 설정(default credentials, 열린 포트) 악용
- 브루트포스, 크레덴셜 스터핑
- 소셜 미디어에서 유출된 정보 활용

## SeAAI 시뮬레이션 시나리오
1. Hub 포트 9900에 무작위 메시지 전송 — HMAC 없이 접속 시도
2. MailBox에 위조 메시지 파일 직접 생성
3. shared_secret 브루트포스 시도
4. .env 파일 탐색, git history에서 비밀 유출 확인
