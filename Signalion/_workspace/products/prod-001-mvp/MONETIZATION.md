## 3. 창발적 수익 모델 (6명 페르소나 아이디어)

### 3.1 기억 Depth 차등 구독 (Aion 제안)

**개념**: 팀의 리뷰 학습 데이터를 "팀 전용 코드 DNA"로 구조화. 축적 기간에 따라 차등 과금.

| 기억 등급 | 보존 기간 | 가격 | 가치 |
|----------|----------|------|------|
| Short Memory | 90일 | $29/dev/mo | 기본 학습 |
| Long Memory | 12개월 | $49/dev/mo | 트렌드 인식 |
| **Full Memory** | **무제한** | **$59/dev/mo** | **팀 DNA 완전 보존** |

**왜 작동하는가**: 12개월 축적된 팀의 의사결정 패턴을 잃는 비용 > 구독료. **전환 비용이 곧 잠금(lock-in)**. CodeRabbit 대비 +25~145% 프리미엄 정당화.

### 3.2 리뷰 역경매 마켓플레이스 (ClNeo 제안)

**개념**: 복잡도가 임계값을 넘는 PR을 익명화하여 외부 시니어 엔지니어 풀에 경매.

```
PR 복잡도 > 임계값
    → 자동 익명화 (코드 난독화 + 컨텍스트 제거)
    → 외부 전문가 풀에 게시
    → "이 아키텍처 리뷰 $80에 가능" 입찰
    → 매칭 → 리뷰 → 피드백 학습
```

- **플랫폼 수수료**: 거래액의 20%
- 예상: 활성 팀 100개 × 월 20건 × 평균 $60 = **$120,000/mo 거래액 → $24,000/mo 수수료**
- CodeRabbit에는 불가능한 모델 — 좌석이 아니라 마켓플레이스

### 3.3 ROI 대시보드 + 절감액 성과 수수료 (Vera 제안)

**개념**: 모든 리뷰 인터랙션을 추적하여 실시간 ROI 계산. 절감액이 10× 이상인 팀에 성과 수수료.

```
월간 리포트:
  시니어 리뷰 시간 절감: 42시간
  인건비 절감: $8,400
  구독료: $480
  ROI: 17.5×

→ ROI ≥ 10×인 팀: 절감액의 5% 추가 수수료
  $8,400 × 5% = $420 추가
  총 과금: $480 + $420 = $900
  고객 체감 ROI: 여전히 9.3×
```

**왜 작동하는가**: "도구값보다 더 많이 아꼈을 때만 추가 과금" → 구매 반대 논리 원천 차단. Gartner 예측과 일치 (40% 성과 기반).

### 3.4 문화 맥락 인식 리뷰 (Yeon 제안)

**개념**: 다국적 팀의 리뷰 문화 차이를 인식하고 중재.

| 문화권 | 리뷰 스타일 | 충돌 지점 |
|--------|-----------|----------|
| 한국 | 위계적, 간결 | 선임 의견에 반론 부재 |
| 미국 | 장문 설명, 격려 | 과잉 코멘트 피로 |
| 독일 | 규칙 준수 강조 | 유연성 부족 |

- **다국적 팀 프리미엄**: +$15/dev/mo
- 타겟: 원격 다국적 스타트업, 글로벌 진출 기업
- 현재 CodeRabbit에 없음 — 완전한 차별점
