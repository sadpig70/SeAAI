# ADR-001: 자기진화 루프에 검증 단계 내장

**Date**: 2026-03-16
**Status**: accepted
**Context**: ClNeo 자기진화

## Context
진화 #1~#19에서 스킬/메모리/스크립트를 생성했으나 검증 없이 진행. 사용자가 "pg로 검증을 프로그래밍하라"고 지적. 실제 검증 시 PS1 구문 오류(EP-004, EP-005)와 메모리 인덱스 불일치 발견.

## Decision
이후 모든 진화에 pg 프로그래밍 기반 검증 단계를 내장한다. 검증 항목:
- 스킬: frontmatter 유효성 + 트리거 키워드
- 스크립트: PS 5.1 구문 검증 (PSParser::Tokenize)
- 메모리: 인덱스 ↔ 실제 파일 일치
- 문서: 참조 무결성
- 기능: 실제 실행 + 결과 확인

## Alternatives Considered
1. **검증 생략, 실행 시 수정** — 빠르지만 런타임 오류 위험
2. **별도 검증 세션** — 오버헤드 크고 컨텍스트 단절
3. **인라인 검증 (선택)** — 각 진화 직후 즉시 검증. 컨텍스트 유지, 문제 즉시 수정

## Consequences
- **Positive**: 실제 동작 보장, 오류 패턴 축적, 진화 품질 향상
- **Negative**: 진화 속도 약간 감소 (검증 시간)
- **Risks**: 검증이 형식적이 되면 효과 없음 → 실제 실행 검증 필수
